# Certifications

A collection of certificates and awards from my computer science journey.

> This repository contains images of certificates and awards I have earned. Each certificate is stored as an image file inside categorized folders.

---

## Structure

The repository is organized into folders (examples):

* `Agent X Certificates` — certificates related to Agent X program
* `Best Mentor IIUCCPS` — mentor recognition
* `Edge (Python Django)` — course/project certificate
* `ICPC 2023 Regional Preliminary` — ICPC participation certificate
* `IIUCCPS Bootcamp Best Performer` — bootcamp recognition
* `NSU WebXtreame Hackathon` — hackathon participation/awards

> Note: All files in these folders are image files (JPEG/PNG). If you add a new certificate, please keep the filename meaningful and add it to the appropriate folder.

---

## How to view

1. Browse the repository on GitHub and open any folder to preview images directly in the browser.
2. Clone the repo locally and open images with your preferred image viewer:

```bash
git clone https://github.com/duttaturja/Certifications.git
cd Certifications
# open images using your OS image viewer or a gallery app
```
